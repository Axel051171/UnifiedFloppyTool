Konvention:
- `<version>-<scope>.md` — z.B. `v4.1.5-m3-hal-wiring.md`
- Abgeschlossene Goals nach `archive/` verschieben (nicht löschen — Audit-Trail)
- Status-Legende: ✅ done · 🟡 in progress · ⬜ pending · 🛑 STOP-Gate
- Branch im Header benennen (NICHT raten lassen)
