# Support

## Reporting Bugs

When filing a bug report, please include:

1. **OS and version** (e.g., Windows 11, Ubuntu 22.04, macOS 14)
2. **UnifiedFloppyTool version** (from Help > About, or the release you downloaded)
3. **Steps to reproduce** the issue
4. **Expected behavior** vs. **actual behavior**
5. **Disk image format** you were working with (if applicable)
6. **Hardware** (if using a physical floppy controller: GreaseWeazle, KryoFlux, etc.)

File reports at: https://github.com/Axel051171/UnifiedFloppyTool/issues

## Questions and Discussions

For general questions, use GitHub Discussions (if enabled) or open an issue
with the **question** label.
