# mbedTLS (Vendor Copy)

This is a vendored copy of mbedTLS used by hactool for Nintendo Switch
crypto operations (AES, SHA, RSA). It is NOT maintained -- security
patches are not applied.

If Switch support is removed (`CONFIG+=switch_support` is not set),
this entire directory can be safely deleted.
