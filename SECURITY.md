# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 4.1.x   | :white_check_mark: |
| < 4.1   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in UnifiedFloppyTool, please report it
responsibly by emailing **ax0511el@gmail.com** with a description of the issue.

Please do **not** open a public GitHub issue for security vulnerabilities.

You can expect an initial response within 7 days.
